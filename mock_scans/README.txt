Демо-источник для MockScanner (работа без реального оборудования).

Используется, пока в конфигурации mock_scanners = 1 (см. armcore/config.py).
Кнопки сканирования в АРМ берут изображения отсюда.

Структура:
  passport/         изображения паспорта (главная страница и т.п.)
  passport/mrz.txt  текст MRZ (2 строки TD3) — подставляется в поля
  pages/            все страницы паспорта (режим booksheet / перевод)
  document/         сшитый многостраничный документ (имитация Kodak)

Если какой-то подпапки нет — MockScanner генерирует пустые цветные листы 300 dpi.
Для реального оборудования: mock_scanners = 0 и реализуйте TODO в armcore/scanners.py.
